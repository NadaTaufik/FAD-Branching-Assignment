# FAD-Branching-Assignment
Branching Statement Assignment (Java)

This project is part of the Fundamentals of Application Development course.

## Description
This assignment demonstrates the use of branching statements in Java:
- if-else
- nested if
- switch-case

## Project Structure

- Part 1: Buying Food (Switch Case)
- Part 2: Grading System (If-Else)
- Part 3: Login Access (Nested If)

## Technologies Used
- Java
- VS Code

## How to Run
1. Open the project in VS Code
2. Run each `.java` file separately
3. Follow the input instructions in the console
